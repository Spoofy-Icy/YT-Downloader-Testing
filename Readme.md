# ⚡ YT Downloader

A simple and modern YouTube video downloader interface built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project demonstrates a frontend interface for downloading YouTube videos using a clean and responsive UI. It is designed as a learning project to understand how video downloading tools work with Python (yt-dlp) and how to present them with a user-friendly interface.

## 🚀 Features

- Paste YouTube video link
- Fetch video preview (thumbnail)
- Display video details
- Animated download button
- Responsive design (mobile + desktop)
- Clean dark UI with gradient styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Python 3.13 and (yt-dlp) Library
- Bootstrap Icons
- Google Fonts
